How to run the app:
1. Clone or download the repository
2. Position inside directory
3. Run one of the following commands
```
    //Python 2.x
    python -m SimpleHTTPServer

    //Python 3.x
    python -m http.server
```
4. App should be running on
```http://localhost:8000/```
